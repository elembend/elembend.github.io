- Information -
Hello, Are you wondering what this whole server is about? Well! You are in the right place.

This is a avatar bending server, If you didn't watch the avatar show then basicly there are 4 main powers; Air, Water, Fire and earth.
Each power has unique abilities.
So the server is a survival server, with these powers and abilities! There are power masters, bases and so much more! We've got a cool and chill community with a lot of friendly people.
